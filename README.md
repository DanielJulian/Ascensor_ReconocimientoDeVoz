Programa realizado en Java 1.7 
Para iniciar el ascensor , debe pronunciar : " Iniciar reconocimiento "

Luego, el ascensor responderá a los comandos que se muestran en el panel. Por ejemplo : " Ir al piso X " , donde X es un número del 1 al 60.
Al pronunciar el comando, se iluminará el piso correspondiente en el panel. Para que el ascensor se mueva, debe clickear en cerrar puertas.
